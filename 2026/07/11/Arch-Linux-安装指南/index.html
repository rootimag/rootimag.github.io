Arch Linux 安装指南

Arch Linux 安装指南

0. 进入安装环境

推荐使用 ventoy 引导，直接将 Arch Linux ISO 拷贝进 U 盘即可。

选择第一个选项。

进入 Arch Linux Live 的 TTY，就可以开始安装了。

01. 确认 UEFI 引导

在正式安装前，需要先确认是否为 UEFI 引导。

ls /sys/firmware/efi/efivars

若显示 efivars,则为 UEFI 引导。

02. 连接网络

有线连接

通常情况下，只要插入由路由器提供的有线网卡即可。

无线连接

Arch Linux Live 内置了 iwd 无线 wifi 模块，建议使用 iwctl 进行无线连接。

测试网络连接:

ping -c4 www.baidu.com

若有数据包传输，则连接成功。

03. 更新系统时钟

使用 timedatectl 确保时钟正确。

05. 更换镜像源

使用 vim 修改 /etc/pacman.d/mirrorlist 文件，将 Server 替换为你需要的镜像源。

vim /etc/pacman.d/mirrorlist

06. 创建与格式化分区

创建分区

考虑到多数情况，本指南将以以下分区为例:

  • / 根目录: >= 120GB （与用户主目录位于同一 btrfs 文件系统上）
  • /home 用户主目录: >= 20GB （与根目录位于同一 btrfs 文件系统上）
  • /boot EFI 引导分区: >= 256MB （若有 Windows 系统，无须重复创建！）
  • swap 交换分区: >= RAM * 60%
  1. 查看分区情况

通过 lsblk 查看分区情况。

sdx / nvmexn1 分区为将要安装系统的空白分区

  1. 建立分区表

通过 cfdisk 创建分区。

cfdisk /dev/sdx # 对于 SCSI 指令集的硬盘

cfdisk /dev/nvmexn1 # 对于 NVMe 指令集的硬盘

对于分区表类型，选择 GPT 类型。

  1. 创建分区

进入 cfdisk 后，使用 ↑ 和 ↓ 在磁盘分区移动，使用 ← 和 → 在分区操作移动。

按照如图所示，创建分区后，选中 [Write] 进行写入。

  1. 检查分区

使用 fdsik -l 或 lsblk 查看分区情况。

格式化分区

  1. 格式化 EFI 引导分区

mkfs.fat -F32 /dev/sdx1

mkfs.fat -F32 /dev/nvmexn1

Warning: 如果硬盘上已有操作系统，请不要格式化，否则会导致无法引导！

  1. 格式化 Swap 分区

mkswap /dev/sdx2

mkswap /dev/nvmexn2

  1. 格式化 / 分区

mkfs.btrfs -L [label] /dev/sdx3

mkfs.btrfs -L [label] /dev/nvmexn3

参数说明：label 为分区标签，请填写你喜欢的名称。

  1. 挂载 / 分区

mount -t btrfs -o compress=zstd /dev/sda3 /mnt

mount -t btrfs -o compress=zstd /dev/nvmexn3 /mnt

参数说明:

  • -t 指定文件系统类型
  • -o 指定文件系统选项
    • compress=zstd 使用 zstd 压缩文件系统
  1. 创建 Btrfs 子卷
  1. 卸载子卷

umount /mnt

07. 挂载分区

在挂载时，需按照如下顺序进行挂载:

  1. 挂载 / 目录

mount -t btrfs -o subvol=@,compress=zstd /dev/sda3 /mnt

  1. 挂载 /home 目录
  1. 挂载 /boot 目录
  1. 挂载 /swap 目录

swapon /dev/sda2

08. 安装基础软件包

通过 pacstrap 安装基础软件包。

pacstrap /mnt base base-devel linux linux-firmware btrfs-progs

参数说明:

  • base 基础软件包
  • base-devel 基础开发软件包
  • linux 内核
  • linux-firmware 内核固件
  • btrfs-progs Btrfs 文件系统工具

安装必要功能性软件包。

pacstrap /mnt networkmanager vim sudo zsh zsh-completions

09. 生成 fstab 文件

fstab 用于定义分区挂载点，以便在系统启动时自动挂载分区，使用 genfstab 生成 fstab 文件。

genfstab -U /mnt >> /mnt/etc/fstab

10. 切换到新系统

arch-chroot /mnt /bin/bash

11. 主机名与时区设置

在 /etc/hostname 设置主机名

在 /etc/hosts 设置主机名解析

加入 127.0.1.1 [name].localdomain [name]

在 /etc/localtime 设置时区

ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

  1. 硬件时间设置

使用 hwclock 设置硬件时间，将系统时间同步到硬件时间，以便在系统重启后不会丢失时间。

hwclock --systohc

  1. locale 设置

locale 决定了系统的语言环境，包括字符集，日期格式，货币符号等。

编辑 /etc/locale.gen 文件，取消 en_US.UTF-8 和 zh_CN.UTF-8 的注释，并保存文件。

vim /etc/locale.gen

运行 locale-gen 生成本地化文件。

向 /etc/locale.conf 添加语言环境变量，以便在系统启动后生效。

echo "LANG=en_US.UTF-8" > /etc/locale.conf

14. root 用户设置

为了便于使用，我们需要设置 root 用户的密码。

passwd root

15. 安装引导

安装引导软件包。

pacman -S grub efibootmgr os-prober

参数说明:

  • grub GRUB 引导
  • efibootmgr EFI 引导
  • os-prober 检测操作系统

安装 GRUB 到 EFI 分区:

grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=ArchLinux

参数说明:

  • --target=x86_64-efi 指定引导目标为 x86_64-efi
  • --efi-directory=/boot 指定 EFI 分区路径
  • --bootloader-id=ArchLinux 指定引导 ID

编辑 /etc/default/grub 文件:

  • GRUB_CMDLINE_LINUX_DEFAULT 改为 loglevel=5 nowatchdog
  • 添加 GRUB_DISABLE_OS_PROBER=false 以引导 windows

生成 GRUB 配置文件:

grub-mkconfig -o /boot/grub/grub.cfg

16. 系统启动

使用 exit 退出 Arch Linux Live 环境，然后重启电脑即可。

设置 networkmanager 服务。

systemctl enable --now NetworkManager.service

然后使用 nmtui 或 nmcli 连接网络

17. 大功告成

使用 pacman -S fastfetch 安装 fastfetch，查看系统信息。

如此，Arch Linux 安装就完成了。
